# Institute of Digital Risk – Homepage

This project is a responsive single-page website created as part of an internship selection assignment. The goal was to design a simple brand identity and build a homepage for the **Institute of Digital Risk (IDR)**, an industry-led training and deployment institute focused on digital, cyber, and AI risk.

## Features

* Responsive homepage design (desktop and mobile)
* Sticky navigation bar with smooth scrolling
* Hero section presenting IDR’s mission
* About section explaining the institute
* Service model section with three pillars:

  * Academy
  * Innovation
  * Advisory
* Community section describing the target audience
* Contact form for registering interest
* Animated statistics counters
* Dark/light theme toggle
* Custom logo design for the IDR brand

## Technologies Used

* HTML5 (semantic structure)
* CSS3 (Flexbox, Grid, responsive design)
* JavaScript (vanilla JS for interactions)
* Google Fonts (Inter)
* Font Awesome icons

## Logo Design

The logo uses a **cube-inspired geometric structure** to represent systems, resilience, and structured risk management. The internal maze-like paths symbolize complex digital ecosystems and cybersecurity networks. The orange accent represents innovation and technological advancement, while black and grey convey stability and professionalism.
